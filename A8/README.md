# Estrutura de Dados — Atividade 8

## Questões

1. **Implementem uma fila de alunos com todas as operações vistas.**

   O aluno deve ter os seguintes dados: nome, cpf, idade, matrícula e uma lista de disciplinas (cadeiras/matéria). Cada disciplina tem nome, quantidade de créditos e uma lista de conteúdo (lista de strings).

2. **Crie as seguintes funções adicionais:**

   - Adicionar uma disciplina a um aluno.  
   - Remover uma disciplina de um aluno.  
   - Listar todas as disciplinas de um aluno.  
   - Adicionar um novo conteúdo a uma disciplina.  
   - Remover um conteúdo de uma disciplina.  
   - Listar os conteúdos de uma disciplina.  
   - Listar todos os alunos, com suas disciplinas e os conteúdos da mesma.
