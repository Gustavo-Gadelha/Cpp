# Estrutura de Dados — Atividade 7 (Pilha)

## Questões

1. **Deletar o elemento do meio de uma pilha**

   Dada uma pilha encadeada, remova o elemento do meio.  

2. **Reverter os elementos de uma pilha usando recursividade**

   Escreva um procedimento recursivo para inverter a pilha sem usar estruturas auxiliares explícitas.  

3. **Reverter uma string utilizando uma pilha**

   Dada uma string, empilhe seus caracteres e desempilhe para produzir a string reversa.  
   Exemplo: `"Estrutura de Dados em cpp"` ⟶ `"ppc me sodaD ed aruturtsE"`.

4. **Pilha de tamanho fixo**

   Implemente uma pilha encadeada com capacidade fixa, contendo:
   - `push(data)`
   - `pop()`
   - `top()/peek()`
   - `isFull()`
